  What I'd Do If I Were You

  Phase 1: Plant the Flag (This Month)

  - Rename/rebrand - "Omni-Roast" or similar (bigger than one store)
  - Add roast profile schema to your app
  - Build a simple profile browser (public page, no auth needed)
  - Seed with 10-20 real profiles you've tried yourself
  - Post in r/roasting, r/coffee, home-roasters.com forums

  Phase 2: Validate the Nerd Market (Next Month)

  - Can you get 100 profiles uploaded? (even manually)
  - Do people actually browse them?
  - Does anyone click "buy these beans"?

  Phase 3: Hardware Integration (When Validated)

  - Artisan Roasting Software - open source, has export format
  - Cropster - commercial but has API
  - Build simple import - "Upload your Artisan log"
